  • Oplysninger

  • Du er økonomikonsulent for en virksomhed, der producerer kontormøbler. Virksomheden ønsker at optimere sin prissætning og har indsamlet data til at estimere prisafsætningsfunktionerne for deres produkter.

    For skrivebordet "Basic" er prisafsætningsfunktionen estimeret til: P = -0,2X + 2000
    hvor P er prisen i kr. og X er antallet af skriveborde pr. måned.

    For skrivebordet "Premium" er prisafsætningsfunktionen estimeret til: P = -0,4X + 5000
    hvor P er prisen i kr. og X er antallet af skriveborde pr. måned.

    For kontorlampen "FlexLight" har virksomheden estimeret omsætningsfunktionen til: Oms = -0,3X² + 1800X
    hvor Oms er den månedlige omsætning i kr. og X er antallet af lamper.

    For kontorstolen "Ergo" har virksomheden estimeret grænseomsætningsfunktionen til: GROMS = -1,2X + 3600
    hvor GROMS er grænseomsætningen i kr. og X er antallet af stole.

    Virksomheden har også indsamlet data om, hvordan prisændringer på deres "Basic" skrivebord påvirker salget af "Premium" skrivebord. Når prisen på "Basic" ændres fra 1000 kr. til 1200 kr., observerer virksomheden typisk en stigning i salget af "Premium" fra 2000 stk. til 2160 stk.

  • Spørgsmål

    1. Hvad er funktionsforskriften for grænseomsætningen (Groms) for skrivebordet "Basic" (P = -0,2X + 2000)?
      Vink: For en lineær prisafsætningsfunktion P = aX + b er grænseomsætningsfunktionen Groms = 2aX + b.
      Løsningsforslag:
      For skrivebordet "Basic" er prisafsætningsfunktionen: P = -0,2X + 2000
      Grænseomsætningsfunktionen er:
      Groms = 2 · (-0,2) · X + 2000 = -0,4X + 2000
      Funktionsforskriften for grænseomsætningen er: Groms = -0,4X + 2000
      Dette ses i forskriften for Groms på den blå label i billedet fra app'en herunder:

    2. Hvad er afsætningen X for skrivebordet "Basic" (P = -0,2X + 2000) når prisen er 1400 kr.?
      Vink: Find afsætningen X når prisen er 1.400 kr. ved at løse ligningen P = -0,2X + 2000 eller benyt app'en.
      Løsningsforslag:
      Først finder vi X-værdien for P = 1400 kr. ved at omskrive prisafsætningsfunktionen:
      P = -0,2X + 2000
      1400 = -0,2X + 2000
      -0,2X = 1400 - 2000
      -0,2X = -600
      X = 3000 stk.

      Afsætningen for skrivebordet "Basic" når prisen er 1400 kr. er 3000 stk.
      Dette ses i på den røde label fra app'en herunder:

    3. Hvad er grænseomsætningen (Groms) for skrivebordet "Basic" (P = -0,2X + 2000) når prisen er 1400 kr.?
      Vink: Du fandt tidligere afsætningen X når prisen er 1.400 kr.
      Brug derefter formlen Groms = 2aX + b, hvor a og b er koefficienterne fra prisafsætningsfunktionen, indsæt afsætningen X og beregn.
      Løsningsforslag:
      Når afsætningen X = 3000 findes er værdien af grænseomsætningen Groms når afsætningen indsættes i grænseomsætningsfunktionen:
      GROMS = -0,4 · 3000 + 2000 = -1200 + 2000 = 800 kr.
      Grænseomsætningen for skrivebordet "Basic" når prisen er 1400 kr. er 800 kr.
    4. Beregn den maksimale omsætning for skrivebordet "Basic" (P = -0,2X + 2000) og ved hvilken pris og mængde denne opnås.
      Vink: Omsætningen er maksimal når grænseomsætningen er lig nul.
      Løsningsforslag:
      Omsætningen er lig med pris gange mængde: Oms = P · X
      Omsætning = P · X = (-0,2X + 2000) · X = -0,2X² + 2000X

      Omsætningen er maksimal når grænseomsætningen er 0:
      GROMS = -0,4X + 2000 = 0
      -0,4X = -2000
      X = 5000 stk.

      Den tilsvarende pris er:
      P = -0,2 · 5000 + 2000 = -1000 + 2000 = 1000 kr.

      Den maksimale omsætning er:
      Oms = 1000 kr. · 5000 stk. = 5.000.000 kr.
      Dette ses på den røde label fra app'en herunder:

    5. Hvad er den maksimale afsætning for skrivebordet "Basic" (P = -0,2X + 2000)?
      Vink: Den maksimale afsætning opnås når prisen er 0 kr.
      Løsningsforslag:
      Den maksimale afsætning finder vi ved den pris, hvor forbrugerne ikke længere er villige til at betale for produktet, dvs. når P = 0:
      P = -0,2X + 2000 = 0
      -0,2X = -2000
      X = 10.000 stk.
      Bemærk vi kan også bruge formlen for skæringen med X-aksen:
      X = -b/a = -2000 / -0,2 = 10.000 stk.

      Den maksimale afsætning for "Basic" skrivebordet er 10.000 stk. pr. måned. Dette ses i på den orange label fra app'en herunder:

    6. Bestem prisen for "Premium" skrivebordet (P = -0,4X + 5000), når grænseomsætningen er 3000 kr.
      Vink: Denne opgave er lidt sværere, her kan man ikke direkte benytte appen.
      Bestem først Groms ud fra P.
      Find nu X-værdien der giver den angivne grænseomsætning ved at løse ligningen hvor Groms = 3000
      Indsæt til slut X-værdien i prisafsætningsfunktionen P.
      Løsningsforslag:
      For "Premium" skrivebordet gælder prisafsætningsfunktionen: P = -0,4X + 5000
      Grænseomsætningsfunktionen er: GROMS = 2 · (-0,4) · X + 5000 = -0,8X + 5000

      Når GROMS = 3000 kr:
      -0,8X + 5000 = 3000
      -0,8X = 3000 - 5000
      -0,8X = -2000
      X = 2500 stk.

      Den tilsvarende pris er:
      P = -0,4 · 2500 + 5000 = -1000 + 5000 = 4000 kr.
      Prisen for "Premium" skrivebordet når grænseomsætningen er 3000 kr. er 4000 kr. Vi kan se i app'en herunder at vi har fundet den korrekte pris, den stiplede røde linje skærer Groms = 3000 kr. hvor X = 2500 stk. og P = 4000 kr.:

    7. For kontorlampen "FlexLight" (Omsætningsfunktion: Oms = -0,3X² + 1800X), udled prisafsætningsfunktionen P og grænseomsætningsfunktionen GROMS baseret på den givne omsætningsfunktion.
      Vink: Denne opgave er igen lidt svær, vi kan ikke benytte app'en direkte.
      Brug at Oms = P·X, og derfor P = Oms/X. Grænseomsætningen er den afledte af omsætningsfunktionen.
      Løsningsforslag:
      Fra opgaven har vi omsætningsfunktionen: Oms = -0,3X² + 1800X

      Prisafsætningsfunktionen finder vi ved P = Oms/X:
      P = Oms/X = (-0,3X² + 1800X)/X = -0,3X + 1800

      Grænseomsætningsfunktionen er den afledte af omsætningsfunktionen:
      GROMS = Oms' = (-0,3X² + 1800X)' = 2 · -0,3X + 1800 = -0,6X + 1800

      Prisafsætningsfunktionen er: P = -0,3X + 1800
      Grænseomsætningsfunktionen er: GROMS = -0,6X + 1800
    8. Beregn den maksimale omsætning for kontorlampen "FlexLight" (Oms = -0,3X² + 1800X) og ved hvilken pris og mængde denne opnås.
      Vink: Omsætningen er maksimal når grænseomsætningen er lig nul, eller du kan bruge at P er lineær, til hurtigt at finde prisen, der maksimerer omsætningen.
      Løsningsforslag:
      Omsætningen er maksimal når grænseomsætningen er 0:
      GROMS = -0,6X + 1800 = 0
      -0,6X = -1800
      X = 3000 stk.

      Den tilsvarende pris er:
      P = -0,3 · 3000 + 1800 = -900 + 1800 = 900 kr.

      Denne pris kunne vi også finde da vi ved at en pris på b/2 = 1.800/2 = 900 kr. giver maksimal omsætning for en lineær funktion.
      Den maksimale omsætning er:
      Oms = 900 kr. · 3000 stk. = 2.700.000 kr.

      Vi kan også beregne den maksimale omsætning direkte ved at indsætte X = 3000 i omsætningsfunktionen:
      Oms = -0,3 · 3000² + 1800 · 3000 = -0,3 · 9.000.000 + 5.400.000 = -2.700.000 + 5.400.000 = 2.700.000 kr.

    9. For kontorstolen "Ergo" (Grænseomsætningsfunktion: GROMS = -1,2X + 3600), udled prisafsætningsfunktionen og omsætningsfunktionen baseret på den givne grænseomsætningsfunktion.
      Vink: For en lineær grænseomsætningsfunktion GROMS = 2aX + b, er prisafsætningsfunktionen P = aX + b.
      Når vi har fundet P kan vi finde omsætningsfunktionen ved at gange P med X.
      Løsningsforslag:
      For kontorstolen "Ergo" har vi grænseomsætningsfunktionen: GROMS = -1,2X + 3600

      For en lineær grænseomsætningsfunktion på formen GROMS = 2aX + b, er prisafsætningsfunktionen P = aX + b.
      Vi kan skrive GROMS = -1,2X + 3600 som GROMS = 2(-0,6X) + 3600
      Det betyder at a = -0,6 og b = 3600

      Prisafsætningsfunktionen er derfor:
      P = -0,6X + 3600

      Omsætningsfunktionen finder vi ved at gange pris og mængde:
      Oms = P · X = (-0,6X + 3600) · X = -0,6X² + 3600X

      Vi har i app'en herunder indsat forskriften for P, under figurerne er en tabel der viser forskriften for omsætningen.

    10. Beregn priselasticiteten for "Basic" skrivebordet (P = -0,2X + 2000) ved en pris på 1600 kr. Forklar resultatets betydning.
      Vink: For en lineær prisafsætningsfunktion P = aX + b er priselasticiteten ε = (1/a) · (P/X).
      Løsningsforslag:
      Først finder vi X-værdien ved P = 1600 kr:
      P = -0,2X + 2000
      1600 = -0,2X + 2000
      -0,2X = 1600 - 2000
      -0,2X = -400
      X = 2000 stk.

      For en lineær prisafsætningsfunktion kan priselasticiteten beregnes ved formlen:
      ε = (dX/dP) · (P/X) = (1/a) · (P/X)

      Her er a = -0,2, så:
      ε = (1/(-0,2)) · (1600/2000) = -5 · 0,8 = -4

      Priselasticiteten er -4, hvilket betyder at efterspørgslen er elastisk.
      Konkret betyder det, at hvis prisen stiger med 1%, vil efterspørgslen falde med 4%.
      Det indikerer, at forbrugerne er meget følsomme over for prisændringer på "Basic" skrivebordet ved denne pris.

      Det ses i app'en herunder, den sorte label viser priselasticiteten ε = -4:
    11. Beregn krydspriselasticiteten mellem "Basic" (P = -0,2X + 2000) og "Premium" (P = -0,4X + 5000) skrivebordene når prisen på "Basic" ændres fra 1000 kr. til 1200 kr., hvilket ændrer salget af "Premium" fra 2000 stk. til 2160 stk. Hvad fortæller resultatet om relationen mellem de to produkter?
      Vink: Brug formlen εAB = (ΔXB/XB)/(ΔPA/PA) hvor XB er mængden af "Premium" og PA er prisen på "Basic". Du kan bruge app'en til krydspriselasticitet.
      Løsningsforslag:
      Krydspriselasticiteten er defineret som den procentvise ændring i efterspørgslen efter vare B divideret med den procentvise ændring i prisen på vare A:
      ε_xy = (ΔY/Y)/(ΔP_x/P_x)

      Vi indsætter de givne værdier:
      ΔP_x = 1200 - 1000 = 200 kr.
      P_x = 1000 kr.
      ΔY = 2160 - 2000 = 160 stk.
      Y = 2000 stk.

      ε_xy = (ΔY/Y)/(ΔP_x/P_x) = (160/2000)/(200/1000) = (0,08)/(0,2) = 0,4

      Krydspriselasticiteten er 0,4.
      Dette indikerer, at "Basic" og "Premium" skrivebordene er substitutter - når prisen på "Basic" stiger, vil nogle kunder skifte til "Premium".
      Da værdien er mellem 0 og 1, er der tale om svage substitutter.
      Det ses i app'en herunder viser krydspriselasticiteten ε = 0,4:

    12. Hvilken pris skal virksomheden sætte for kontorstolen "Ergo" (GROMS = -1,2X + 3600) for at opnå den maksimale omsætning?
      Vink: Omsætningen er maksimal når grænseomsætningen er lig nul.
      Løsningsforslag:
      Vi ved at grænseomsætningsfunktionen for kontorstolen "Ergo" er: GROMS = -1,2X + 3600

      Omsætningen er maksimal når grænseomsætningen er 0:
      -1,2X + 3600 = 0
      -1,2X = -3600
      X = 3000 stk.

      Ved at indsætte X = 3000 i prisafsætningsfunktionen P = -0,6X + 3600, får vi:
      P = -0,6 · 3000 + 3600 = -1800 + 3600 = 1800 kr.

      Virksomheden skal sætte prisen til 1800 kr. pr. stol for at opnå den maksimale omsætning.
      Her kunne vi hurtigere have fundet prisen ved formlen P = b/2 = 3600/2 = 1.800 kr.
